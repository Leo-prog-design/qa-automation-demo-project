# QA Automation Demo Project

## Stack

* Python
* Pytest
* Selenium
* Requests
* GitHub Actions

## Project Features

* UI automation testing
* API testing
* Page Object Model
* Test data separation
* CI pipeline

## Running tests

Install dependencies

pip install -r requirements.txt

Run tests

pytest

## CI/CD

Тесты автоматически запускаются при каждом push
